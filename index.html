Farmbot is a new technologhy robot build to help plants and vegetables grow. The Farmbot Genesis is an open soucre hardware machine. The Farmbot Genesis is able to plant over 30 different crops. The farmbot Genesis is also able to gather data about plants; like age and local weather conditions.

Here's alittle Farmbot information video of how it works, you can see that the machine is using a weed suppressor in the ground based on data information.

The project above is an example of how a Farmbot looks. The Farmbot Genesis perform different tasks automatically using different attached tools such as a seed injector, water noozle and a tool to bury seeds.
